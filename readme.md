# Hahow 影片取得

取得 Hahow 的影片與字幕 ( 利用 Hahow API ) ,  
之後可批次上傳到私人 youtube ,  
製作 youtube 撥放清單 ,  
然後在 APP 中進行觀看  

## 桌面 APP 

目前完整 Electron App 在 vue-proj 資料夾中

## 里程碑

1. 特定網址 , 抓出 hahow 上的影片
2. 上傳到私人 youtube 影片
3. 爬蟲整批課程
