# vue-proj
 
  這是一個用 vue 建立 electron 的一個專案

## 安裝步驟 

```bash=
npm install -g @vue/cli
vue add electron-builder
```
然後 , 就可以用 `npm run electron:serve`

## 其他

> 如果想要在 electron-vue 中使用 Node-API 可以參考 `src/preload.js`
